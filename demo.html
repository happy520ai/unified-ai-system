组件化示范

这个页面展示了如何用共享组件构建页面,代码量比传统方式减少 80%

✅ 已完成:共享组件

以下 3 个组件文件取代了每个页面中重复的 sidebar/nav/layout 代码:

<script src="components/nav.js"></script>    <!-- 导航配置 -->
<script src="components/sidebar.js"></script> <!-- 侧边栏渲染 + 交互 -->
<script src="components/layout.js"></script>  <!-- 页面布局骨架 -->

// 一行代码注入侧边栏
injectSidebar();

📊 改造效果对比

以 index.html 为例:

指标 改造前 改造后
Sidebar HTML ~60 行内嵌 0 行 (组件注入)
导航配置 每个页面重复 nav.js 统一管理
Sidebar 折叠逻辑 每个页面重复 JS sidebar.js 自动绑定
新增页面工作量 复制整个 HTML + 改 只写业务内容

🔧 使用方式

新页面只需这样写:

<body>
  <div class="page-wrapper">
    <!-- Sidebar 由组件自动注入 -->
    <div class="page-main" id="page-main">
      <!-- 你的页面内容 -->
    </div>
  </div>

  <script src="i18n.js"></script>
  <script src="interact.js"></script>
  <script src="components/nav.js"></script>
  <script src="components/sidebar.js"></script>
  <script>injectSidebar();</script>
</body>
