# Phase587 Mission Control Internal Communication Preview

Mission Control now includes an Internal Employee Communication Bus preview
after the Workforce Preview panel.

The panel shows:

- Internal Employee Communication Bus
- Employee Threads
- Inbox / Outbox
- Mentions
- Handoff
- Review Requests
- Evidence Timeline
- no-provider-call / no-secret / no-external-IM-send badges

Buttons:

- Preview internal employee thread.
- Preview employee mention collaboration.
- Preview handoff.
- Preview objection.
- Preview council summary.

All buttons update a visible result panel. No button calls a Provider or sends
external IM.
