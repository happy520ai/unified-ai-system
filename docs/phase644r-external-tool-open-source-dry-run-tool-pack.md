# Phase644R External Tool Open Source Dry-Run Tool Pack

## Product Position

This tool pack presents Codex Context Gateway as an external Codex relay and
token-saving engineering helper. It is intended for local dry-run preview,
context-pack inspection, relevant-file scoping, stale gate checks, token budget
checks, and evidence review.

## Default Mode

- external Codex relay / token-saving tool
- local preview / dry-run
- no real Provider required
- no auth.json read
- no Codex config write
- no `/chat` integration
- no `/chat-gateway/execute` integration
- no provider runtime mutation
- no production-ready claim
- no release-ready claim

## Mandatory Entry

Phase632 preflight is the mandatory entry before using this external tool pack.
