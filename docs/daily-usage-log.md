## 使用方式

1. 每天打开 `/chat`，用它做一件真实的事（写代码、查资料、整理文档）
2. 花 2 分钟在上面记录
3. 一周后回顾，真实痛点会自动浮现
4. 把痛点给 agent，让它修——那才是该修的东西
