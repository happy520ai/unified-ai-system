# Phase579 Employee Library Generator Pyramid Expansion

Phase579 generates virtual expert employees from the source-backed Position Library.

Rules:
- A position is not an employee.
- An employee is a schedulable virtual role instance.
- Every generated employee is dry-run by default.
- Brain binding mode remains `dry_run`.
- Provider calls remain blocked.

The generated catalog includes L0-L6 pyramid policies and domain chief generation.
