# Phase576A Employee Pyramid Architecture

Employee Library contains virtual expert role instances mapped to positions. Employees are not real people and are not equivalent to a complete occupation dataset.

## Pyramid Levels

- L0 System Governor
- L1 Executive Council
- L2 Domain Chiefs
- L3 Principal Experts
- L4 Senior Specialists
- L5 Operators
- L6 Assistants

## Runtime Boundary

Employee instances can be selected by a dry-run scheduler. They cannot independently call providers, access secrets, deploy, bill, or generate invoices.
