# Phase577 Occupation Source Governance

Official source registry:
- O*NET: U.S. occupation tasks, skills, knowledge, and abilities.
- SOC: U.S. Standard Occupational Classification hierarchy.
- ISCO: International Standard Classification of Occupations.
- ESCO: European skills, competences, qualifications, and occupations.

The registry is source-backed but preview-only. Each source requires version, license, and sourceRef review before real import.
