# Owner Report Design Contract

phaseRange: Phase1821-1840AIO

## Contract

- 报告比 UI 更重要.
- 标题必须中文.
- 正文只保留 owner 能理解的信息.
- 高级信息放底部.
- 下一步只能给一个主动作.
- 不把 evidence path 放在正文第一屏.

## Required Report Sections

1. 今日小天系统检查报告
2. 一、今天完成了什么
3. 二、发现了什么问题
4. 三、下一步我该做什么
5. 四、高级信息

## Report Copy Rules

- Use direct Chinese.
- Do not say "verifier passed" to owner; say "系统检查通过".
- Do not say "Provider Gate"; say "真实模型调用仍处于保护状态".
- Do not make owner choose between many next steps.
- Put logs, screenshots, verifier result paths, and evidence paths under 高级信息.

## Acceptance Notes

The report can be Markdown or HTML. HTML must use local CSS only, no remote fonts, no CDN, and no external images.
