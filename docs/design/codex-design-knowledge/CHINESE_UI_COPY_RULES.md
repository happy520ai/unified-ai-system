# Chinese UI Copy Rules

phaseRange: Phase1821-1840AIO

## Rules

- 中文优先.
- 不使用 owner 不懂的工程词.
- 如必须出现工程词，必须有中文解释.
- 不写 "验证器通过" 给 owner，而写 "系统检查通过".
- 不写 "Provider Gate"，而写 "真实模型调用仍处于保护状态".
- 不写 "evidence path"，而写 "详细记录已保存，可在高级模式查看".
- 不写 "trace"，而写 "本地检查记录".
- 不写 "dry-run"，而写 "本地预演检查".
- 不写 "phase"，而写 "当前检查阶段" only in Advanced Mode.

## Owner-Facing Voice

- Short.
- Practical.
- Calm.
- One action at a time.
- No English unless it is a fixed product name or advanced technical reference.

## Copy Checklist

- Does the owner know what to click next?
- Does the copy explain whether real Provider calls happened?
- Does every disabled or gated control explain why?
- Does every button click produce feedback?
- Does the first screen avoid phase jargon?
