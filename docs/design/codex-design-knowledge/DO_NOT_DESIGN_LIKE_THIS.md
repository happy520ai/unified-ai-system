# Do Not Design Like This

phaseRange: Phase1821-1840AIO

## Anti-Patterns

- 不做工程后台首页.
- 不做按钮墙.
- 不把模块平铺给 owner.
- 不让 owner 理解 Phase.
- 不把 evidence / trace / JSON 路径放到首页.
- 不做花哨 3D.
- 不恢复依依 / Character.
- 不使用远程字体 / CDN / 外链图片.
- 不把设计稿说成真实可用.
- 不把自动化测试说成人工反馈.
- 不把 local self-use 说成 production-ready.
- 不用漂亮截图替代真实可点击链路.
- 不把 Provider Gate 暴露成 owner 必须理解的词.

## Regression Smells

- More than one owner primary CTA on the first screen.
- Three result cards are missing or renamed into engineering modules.
- Advanced Mode is open by default.
- Button click gives no feedback.
- Report opens but owner cannot see what was completed, what is wrong, or what to do next.
