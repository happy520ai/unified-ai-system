## 入口判定规则

- 真实执行: 能在本地直接运行并产生真实状态、真实文件或真实 HTTP 响应。
- preview / dry-run: 只给出计划、诊断、模拟结果或只读面板。
- gated: 代码入口存在，但必须满足 owner approval、CredentialRef-only、限额与 secret 安全条件后才能进入下一步。
