# Phase650R External Tool Known Limits

- External tool mode is not main-chain integration.
- It is not `/chat` integration.
- It is not `/chat-gateway/execute` integration.
- It is not provider runtime integration.
- It does not call Providers by default.
- It does not read secrets or auth.json.
- It does not write Codex config.
- It does not prove production readiness.
- It does not prove release readiness.
- Token saving is estimated, not a real billing claim.
