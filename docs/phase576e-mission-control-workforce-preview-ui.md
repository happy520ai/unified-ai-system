# Phase576E Mission Control Workforce Preview UI

## Goal

Phase576E adds a secondary Workforce Preview area inside Mission Control. The sample dry-run first-screen entry remains the primary path and the sample result stays hidden until the user clicks start.

## UI Content

The preview shows:

- Position Library source-backed status.
- Employee Pyramid levels.
- Dry-run scheduler policy.
- Brain Adapter boundary.
- no-provider-call / no-secret / no-deploy / no-billing / no-invoice badges.

## Actions

- Preview Employee Pyramid.
- Preview Position Library seed.
- Run Workforce dry-run.

Every action provides a visible response or focus change. This preview does not call providers and does not restore any character module.
