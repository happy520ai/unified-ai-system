# Phase648R External Tool Evidence Dashboard Pack

Phase648R adds a read-only evidence dashboard pack for the external Codex relay/token-saving tool. The Mission Control preview is status-only.

## Read-Only Dashboard Fields

- externalToolMode=true
- Phase632 preflight status
- latest context pack status
- relevant files status
- token budget status
- stale gate status
- nightly runner status
- fallback launcher status
- latest evidence paths
- not `/chat` integrated
- not `/chat-gateway/execute` integrated
- not provider runtime
- productionReady=false
- releaseReady=false

## Forbidden UI Actions

- No execution button.
- No Provider call button.
- No `/chat` integration button.
- No deploy, release, or push button.
- No secret or auth.json input.
