# Phase577 Source Versioning And License Boundary

Version policy:
- Mutable latest aliases are not allowed for sealed imports.
- Every import must pin `sourceVersion`.
- Every import must record `retrievedAt`, `sourceRef`, and rollback evidence.

License boundary:
- Commercial-use review is required before bulk import or redistribution.
- Attribution must be retained.
- Raw bulk redistribution is blocked until reviewed.

Current action is local manifest planning only.
