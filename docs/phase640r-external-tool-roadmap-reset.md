# Phase640R External Tool Roadmap Reset

## Allowed Roadmap

- external tool hardening
- token-saving enforcement
- nightly safe runner
- fallback/manual runner
- CLI wrapper
- operator panel
- evidence dashboard
- open-source dry-run docs

## Not Recommended Under Current Direction

- main-chain integration
- `/chat` wiring
- `/chat-gateway/execute` wiring
- provider runtime mutation
- production rollout

## Next Product Shape

The next useful work should make the external relay tool easier to run,
verify, package, and explain without turning it into production gateway traffic.
