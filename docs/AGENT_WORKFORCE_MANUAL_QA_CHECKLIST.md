# Agent Workforce Manual QA Checklist

Use this checklist for human acceptance of the preview console.

- [ ] UI opens.
- [ ] Template selection is visible.
- [ ] Plan generation works for a demo goal.
- [ ] Role tiers are visible.
- [ ] Clarification questions are visible.
- [ ] Consensus preview is visible.
- [ ] Review package is visible.
- [ ] Approval preview is visible and is not execution approval.
- [ ] OMX handoff preview is visible and is task package text only.
- [ ] Execution readiness blocked is visible.
- [ ] External runner disabled is visible.
- [ ] Save/history shows plan id, selected template, createdAt, plan state, and execution disabled.
- [ ] Export JSON/Markdown works.
- [ ] No real execution happens.

Current boundary: no real Agent execution, no oh-my-codex call, no worktree
creation, no workflow run handoff, and no external runner dispatch.
