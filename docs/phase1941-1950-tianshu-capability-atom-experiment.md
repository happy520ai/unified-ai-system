# Phase1941-1950 Tianshu Capability Atom Experiment

This phase adds a low-risk experimental readout layer for Tianshu Mode.

Capability Atoms are descriptive records only. They contain schema, contracts, dependencies, risk metadata, evidence refs, and rollback notes. They do not contain executable source code and must not execute arbitrary code.

Allowed outcome:
- `tianshuCapabilityReadoutExperimentReady=true`

Forbidden claims:
- Provider stability passed
- Production ready
- Commercial ready
- Owner pilot passed
- P0 blockers cleared
