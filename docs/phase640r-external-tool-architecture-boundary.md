# Phase640R External Tool Architecture Boundary

## Retained Capabilities

- context pack
- relevant files
- token budget
- stale gate
- forbidden full repo scan
- output budget
- crs model_provider guarded path
- evidence ledger
- nightly safe runner / fallback runner

## Frozen Directions

- main-chain integration
- default `/chat` integration
- `/chat-gateway/execute` integration
- provider runtime mutation
- production traffic routing

## Boundary

The external tool can prepare compact context, prompt packs, read-scope plans,
token-saving reports, and operator evidence. It must not become a Provider
runtime path without a future explicit user request and a new approval phase.
